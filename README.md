# NBD-Bass 統合モデル: パチンコ参加者動態の閾値効果補正

Bass 拡張モデル（3状態 ODE）に **NBD（Negative Binomial Distribution）** を組み合わせ、
「年1回以上プレイ」という閾値観測による歪みを補正したベイズ推定フレームワーク。

## 背景

従来の `Pachinko_3State_P1tP2t` では「パチンコ参加人口」の年次推移を直接観測として用いたが、p₁ と p₂ の識別性が不十分だった。

**仮説**: 「年1回以上」の閾値は、**真の離脱**と**低頻度化による閾値下落（NBD 閾値効果）**を区別できない。見かけの p₂ は両者の合成である。

## モデル

個人の年間遊戯回数 $X_i$:

$$X_i \mid \lambda_i \sim \text{Poisson}(\lambda_i), \quad \lambda_i \sim \text{Gamma}(r(t), \alpha(t))$$

観測された参加人口:

$$A_{\text{obs}}(t) = M \cdot a(t) \cdot \underbrace{\left(1 - \left(\frac{\alpha(t)}{\alpha(t)+1}\right)^{r(t)}\right)}_{\text{NBD 閾値補正}}$$

頻度分布（n=1500 の 5 ビン）:

$$(n_1, ..., n_5) \sim \text{Multinomial}(1500, \mathbf{p}_{\text{bin}}(r(t), \alpha(t)))$$

時変 NBD パラメータ:

$$\log r(t) = r_0 + r_1 (t - 2013), \quad \log \alpha(t) = a_0 + a_1 (t - 2013)$$

Bass 3状態 ODE（q₂=0 固定、p₁(t), p₂(t) ガウス摂動）。

## 主要結果（統合モデル）

### Bass 部分
| パラメータ | 推定値 | 95% CI | 解釈 |
|---|---|---|---|
| **M** | 3846 万人 | [3180, 4620] | 総潜在市場（λ=0 含む） |
| **c₂** | **2014.03** | ±4 年 | スマホ期の構造変化点 |
| **p₂_base** | 0.156 | CI/med 0.69 ○ | 平常離脱率（従来から識別改善） |
| A₂ | 0.149 | CI/med 1.15 △ | c₂ 付近の離脱ブースト |
| q₁ | 0.76 | CI/med 1.46 △ | 内因伝播 |
| α (Bass) | 0.24 | CI/med 1.49 △ | 再加入率 |

### NBD 部分
| パラメータ | 推定値 | 95% CI | 結論 |
|---|---|---|---|
| r(2013) | 0.369 | tight | heterogeneity |
| r₁ | 0.008 | [-0.006, 0.022] | ゼロ含む → **不変** |
| α_nbd(2013) | 0.00566 | tight | rate scale |
| a₁_nbd | **-0.022** | [-0.038, -0.006] | **有意減少 ✓** |

### 構造的発見

1. **r は不変**: 頻度分布の形（heterogeneity）は 2008-2019 で変化なし
2. **α_nbd は有意に減少**: 年率 -2.2% log units
3. **E[λ] = r/α は増加**: 2008年 ~59回/年 → 2019年 ~74回/年
4. 「**薄い層が抜けて、残った人は heavy/middle 中心の濃いプレイヤー**」という市場構造
5. **c₂ = 2014 はロバスト**: どのモデル（従来 Bass, NBD 単独, 統合）でも一致

- パチンコ参加人口: レジャー白書（1994-2019年次）

## 一般的枠組みとしての意義

本手法は、**Bass 系モデルへ NBD 観測補正を組み込む一般的フレームワーク**として、以下の領域に拡張可能:

- フィットネスサブスク（月1回以上利用を閾値とする場合）
- SNS 利用（週1回以上アクティブユーザー）
- 遊戯・娯楽習慣全般
- 閾値観測を伴う行動データ全般
